# Nova 10-Pass Concept Iteration

## Pass 1 - Wedge Dock
- Analysis: Start with an aggressive wedge-shell architecture and command-strip login fusion.
- Improvement Plan: Test whether angled shells improve identity without hurting scan flow.

## Pass 2 - Floating Deck
- Analysis: Shift into floating deck cards with compact rail labels and less container bulk.
- Improvement Plan: Check if card layering is clearer than hard panel borders.

## Pass 3 - Stage + Drawer
- Analysis: Use a stage metaphor: primary canvas + collapsible utility drawer + action runway.
- Improvement Plan: Validate launch/read/edit actions in one horizontal flow.

## Pass 4 - Hybrid Fusion
- Analysis: Merge strongest pieces from first 3 passes into a hybrid shell.
- Improvement Plan: Increase information hierarchy and remove redundant labels.

## Pass 5 - Hub Compression
- Analysis: Compress auth and update into one workspace; keep register as mode, not a disconnected screen.
- Improvement Plan: Reduce clicks while keeping full function parity.

## Pass 6 - Roster Expansion
- Analysis: Expand roster handling and make graph controls more explicit for offline saves.
- Improvement Plan: Improve capacity and legibility in character selection.

## Pass 7 - Control Discipline
- Analysis: Normalize control grouping: mode chips, action strip, and consistent side utilities.
- Improvement Plan: Lower cognitive overhead while preserving distinct style.

## Pass 8 - Typography Pass
- Analysis: Tune text scale and line breaks; remove cramped inspector/status copy.
- Improvement Plan: Increase readability and reduce clipping risks.

## Pass 9 - Balance Pass
- Analysis: Rebalance whitespace and panel weight to avoid crowding at 1366x768.
- Improvement Plan: Land near-final arrangement for review.

## Pass 10 - Presentation Candidate
- Analysis: Finalize the strongest variant with stable hierarchy and minimal redundancy.
- Improvement Plan: Prepare for implementation decision.
