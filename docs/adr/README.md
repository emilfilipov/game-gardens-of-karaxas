# Architecture Decision Records (ADR)

This directory tracks architecture decisions for Ambitions of Peace.

Conventions:
- Files are numbered in decision order: `0001-*`, `0002-*`, etc.
- Status values: `proposed`, `accepted`, `superseded`.
- Canonical technical references must link back to accepted ADRs.
